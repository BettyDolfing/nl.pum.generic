nl.pum.generic
==============

Generic extension for civicrm

Using API for creation of:
 - Option Groups
 - Custom Field Groups and -Fields
 - Tags
 - Contact Types
 - Case Types

Using Managed Entities for:
 - Groups
 - Relationship Types
